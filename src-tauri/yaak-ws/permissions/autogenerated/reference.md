## Default Permission

Default permissions for the plugin

- `allow-close`
- `allow-connect`
- `allow-delete-connection`
- `allow-delete-connections`
- `allow-delete-request`
- `allow-duplicate-request`
- `allow-list-connections`
- `allow-list-events`
- `allow-list-requests`
- `allow-send`
- `allow-upsert-request`
